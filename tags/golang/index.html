Creating CLI With Golang

2020-06-15 :: Krishna
#cli  #concurrency  #golang  #github  #goroutines 
This post shows you how to create a cli (Command Line Interface) in golang. We are gonna use Github’s public API to create a github-cli, which shows you info about users. We are not gonna use any third party packages. source code for this post https://github.com/M-krishna/github-cli Introduction The goal here is to capture the usernames from the argument and iterate over it until the data fetched from the server. We will make the fetching process concurrent by using Goroutines.
